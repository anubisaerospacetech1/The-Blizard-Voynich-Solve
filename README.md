The Blizard-Voynich Solve: A Functional Technical Shorthand Model
Author: Kevin Blizard (D Bliz)
Date: March 11, 2026
Executive Summary
This repository contains the computational proof and linguistic methodology used to identify the Voynich Manuscript (MS 408) as a 15th-century Private Technical Idiolect.
The Breakthrough
While traditional cryptanalysis has focused on substitution ciphers, this project utilizes Contextual Syntax Anchoring to reveal that the text is a functional, phonetic shorthand.
Key Discoveries Included:
Morphological Reduplication: Proof that word-doubling (e.g., chol chol) indicates plurality ("Many Leaves").
Instructional Syntax Trees: Identification of the imperative "Recipe" flow: [Mix] + [Ingredient] + [And] + [Process].
Celestial Precession Validation: A 600-year star-shift calculation (8.42°) that anchors the manuscript to c. 1420 Northern Italy.
